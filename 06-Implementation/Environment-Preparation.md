# Environment Preparation

## Purpose

This document records the preparation of the Ubuntu environment before deploying the Nextcloud application stack.

---

## Environment

| Component | Value |
|-----------|-------|
| Platform | VMware Workstation Pro |
| Operating System | Ubuntu Desktop 24.04 LTS |
| Deployment Type | Single Virtual Machine |
| Network | NAT |

---

## Preparation Activities

The operating system was prepared by:

- Updating package repositories.
- Installing the latest security updates.
- Removing obsolete packages.
- Verifying internet connectivity.
- Installing essential administration tools.

---

## Outcome

The operating system was successfully prepared for application deployment and is ready for the installation of PostgreSQL, PHP, Nginx, and Nextcloud.
